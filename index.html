IfMan & RCurtain

OpenSource Java Ruby Rails Redis Heroku Material Design

IfMan is a GUI (Graphical User Interface) for Curtain, that helps the management of new features controlling for users.

It works with simple data structures using Redis. With this, you can set a percentage to your feature to control it, such as "every time that someone login, it has 50% of chance to login using my new service". Or you can set specific values, such as "The user A, B and C can see that feature". It's perfect for A/B testing or paid features.

Authors: Andre Protti, Breno Oliveira, Carlos Menezes, Danillo Souza and Gabriel Queiroz.
Articles: IFMAN your super-hero for Features and A/B testing (PT-BR).
